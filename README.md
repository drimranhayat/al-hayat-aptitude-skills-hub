# Al-Hayat Aptitude Skills Hub

A clean, data-driven aptitude preparation website for FAST, NTS, GAT, HAT, scholarships, jobs, and competitive exams.

## Upload instructions

1. Delete old mixed files/folders from the repository.
2. Upload all files and folders from this package to the repository root.
3. Commit and push to `main`.
4. Open: `https://drimranhayat.github.io/al-hayat-aptitude-skills-hub/`
5. Press `Ctrl + Shift + R` after deployment.

## Clean structure

- `assets/css/style.css` — complete responsive UI
- `assets/js/` — clean aptitude-only JavaScript
- `data/` — data-driven content and question banks
- `practice/` — practice engine page
- `mock-tests/` — timed mock test page
- `progress/` — local progress dashboard
- `skills/` — skill learning pages
- `resources/` — study resources
- `search/` — site search
- `tools/` — exam tools

No VSA/trading/forex/PipFury files are included.
