# PipFury Step 4 — VSA Case Files

This package adds a professional VSA case-file library.

Files included:
- data/vsa-case-files.json — 100 educational VSA case files
- case-files/index.html — updated case file page
- assets/js/case-files.js — rendering, search, market/type filters

These are educational scenarios, not signals or financial advice.

TradingView images are not copied. Add your own marked TradingView snapshot URLs later in the `imageUrl` field and keep TradingView attribution visible.
