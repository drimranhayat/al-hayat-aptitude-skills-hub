# PipFury Step 3 — Practice Engine

This update adds a large VSA practice engine.

## Files included

- `data/vsa-quizzes.json` — 500 VSA MCQs with explanations, chart scenarios, and clear strategy notes.
- `data/vsa-drills.json` — 200 chart-reading drills for commentary, case review, mistake correction, and invalidation practice.
- `practice/index.html` — upgraded practice page.
- `assets/js/practice-engine.js` — renders quizzes/drills with filters and clickable answers.

## Teaching method

Every practice item follows a simple sequence:

1. Background first
2. Level second
3. Signal bar third
4. Confirmation fourth
5. Invalidation fifth

No copied chart images are included. Use your own TradingView snapshots later and keep TradingView attribution visible.
